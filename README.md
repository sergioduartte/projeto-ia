# Projeto - Inteligência Artificial
Projeto da disciplina de Inteligência Artificial Turma 02 - UFCG 2025.2
## Tema do projeto
Implementar busca pela menor distância entre cidades usando algoritmo de Dijkstra.  
Um desafio que pode contar pontos extras se for implementado é usar prints do Google Maps como entrada, fazer o processamento da imagem e extrair o grafo com distâncias. Ou usar informações reais de distâncias entre cidades.
